# Cook Now MVP

## Overview
Turn saved recipes + fridge ingredients into meals users can actually make.

## Tech Stack
- FlutterFlow (UI / mobile)
- Python (backend logic prototype)
- RevenueCat (subscriptions)
- JSON dataset (sample recipes)

## Screens
- Onboarding
- Add Ingredients
- Recipe Results
- Recipe Detail
- Grocery List
- Paywall

## Next Steps / Roadmap
- Connect Python backend for dynamic recipe search
- User login/authentication
- Video / cookbook parsing
- Store-specific grocery recommendations
